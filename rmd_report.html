Superstore Sales Analysis

Introduction

This project analysis Superstore sales data using R programming. The analysis includes sales trend, customer performance, category-wise analysis, regional analysis, profit analysis, and a 12-month sales forecast using the ARIMA model.

## Data Overview The Superstore dataset contains 10,194 records and 21 variables. It includes information about order, customers, products, sales, profit, discounts, shipping details, and regions. The dataset is used to analyze business performance and identify sales trends.

## Data Cleaning The dataset was imported R and checked for missing values and data types. The Order.Date column was converted into Date format to enable time series analysis. The data was then prepared for visualization and forecasting.

## Monthly sales Trend The monthly sales trend shows how total sales changed over time. This analysis helps identify seasonal patterns, business growth, and fluctuations in sales performance.

 # convert order Date to Date format---
  
  data$Order.Date <- as.Date(data$Order.Date, format ="%m/%d/%Y")
  head(data$Order.Date)
## [1] "2023-01-03" "2023-01-04" "2023-01-04" "2023-01-04" "2023-01-05"
## [6] "2023-01-06"
  # Monthly Sales Trend---
  
  monthly_sales <- data %>% mutate (Month =
                                  format(Order.Date,"%y-%m"))%>%
    group_by(Month) %>%
    summarise(Total_Sales = sum(Sales), .Groups = "drop")
  
  ggplot(monthly_sales, aes(x = Month, y = Total_Sales, group = 1)) +
    geom_line(color = "blue", linewidth = 1) + geom_point(color = "red",
                                                          size = 2) +
    labs(title = "Monthly Sales Trend",x = "Month",
         y = "Total Sales") + 
    scale_x_discrete(breaks = monthly_sales$Month[seq(1,
                                                      nrow(monthly_sales),
                                                      by = 3)])+
    theme_minimal() +
    theme(axis.text.x = element_text(angle = 45, hjust = 1))
